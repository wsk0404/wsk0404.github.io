• 熟悉 Linux 的系统常用命令，对 shell 脚本有一定的读写能力

• 熟悉 Vue 、ElementPlus 等前端开发技术

• 熟悉 IDEA 、Git 、Maven 、Apipost 等开发及版本控制 工具

工作经历

郑州灵慧软件科技有限公司 2022.8 -至今

Java 后端 工程师 | 计算机软件

• 负责核心业务模块的 Java 后端开发工作，编写高效、可维护的代码，提升系统整体性能

• 参与系统架构设计，优化微服务架构

• 对自身负责项目模块代码进行维护，进行性能优化

• 参与关键功能模块开发，确保项目按时交付并满足业务需求

• 负责解决开发过程中遇到的技术难题

项目经历

悦护云台

技术 框架 ：Spring Cloud 、MyBatis -Plus 、MySQL 、Redis 、RabbitMQ 、Elasticsearch 、Nacos 、Seata 、支付宝 API 、

Docker 、K8s

项目 描述 ：

悦护云台 是为一家 个护类制造工厂 开发的项目。 为了进行 转型，打造了自营电商平台， 实现生产加销售一体化的

模式 ，打通生产端、库存端与销售端， 提升产品流转效率、库存匹配能力与经营灵活性， 帮助企业实现战略升级 。按

业务边界拆分为用户、商品、库存、购物车、订单、支付、优惠券、秒杀等模块。

职责 描述 ：• 参与 系统架构设计，设计合理的数据库；

• 商品服务对商品进行 Spu ，Sku 进行设计，还有品牌，分类等属性 ；

• 订单服务的开发，实现订单创建、支付确认、取消等功能 ；

• 秒杀服务的 开发 ，利用 Redis 原子操作 + Lua 脚本 解决秒杀场景下的高并发与超卖问题 ；

• 优惠券服务的开发，实现优惠券模板管理、发放、领取，支持多种促销策略灵活配置 ；

极仓 WMS

技术 框架 ：Spring Boot 、MyBatis 、MySQL 、Redis 、MinIO 、Swagger 、Spring Security 、JWT

项目 描述 ：

极仓 WMS 是一款面向制造业 的仓储管理系统，解决传统仓储中作业环节分散、库存数据滞后、人工差错率高等问

题。 系统以 条码 /RFID 技术 为核心， 实现物料与货位的精准识别与实时跟踪。 主要模块包括 到货、 入库、 调拨、 出库、

盘点、库存预警、权限控制。

职责 描述 ：

• 负责数据库表结构设计与优化，涵盖物料、批次、库存、货位、到货 /入库单据等核心实体 ；

• 到货模块 开发 ，实现供应商到货登记、 条码 /RFID 扫描识别、 物料批次校验及质检任务触发， 确保到货信息准确入库 ；

• 入库模块的 开发 ，涵盖货位分配、上架任务调度与库存更新逻辑， 对数据进行保持一致 ；

• 使用 MinIO 文件存储 管理入库单据、质检报告及操作附件，实现文档统一化管理 ；

趣行旅游

技术 框架 ：Spring Cloud 、Spring Boot 、MySQL 、MyBatis -Plus 、Redis 、Elasticsearch 、Spring Security 、JWT

项目 描述 ：

趣行旅游 是一款为旅游机构 的运营 工具 ，其内含有主题路线、本地体验、行程规划等多种服务 。系统通过整合城市

旅游资源、特色活动、文化体验、交通方案等，实现数字化的管理， 提高运营效率和游客满意度，帮助旅游服务机构打

造国内旅游综合服务生态。

职责 描述 ：

• 参与系统架构设计，客户端需求分析以及数据库设计 ；

• 采用 微信登录 + 后端鉴权 的组合方式，实现用户身份识别与业务系统账户绑定 ；

• 产品管理模块的开发， 国内旅游线路、 主题体验及特色活动的管理， 包括产品排期、 可售名额、 价格策略及套餐组合 ；

• 负责游客在线预订及后台订单管理，订单状态管理、取消和退款流程，保证交易完整，同时通过库存校验和异步处理

提升高峰期预订性能和订单转化效率 ；

博睿 CRM

技术 框架 ：Spring Cloud 、Spring Boot 、MySQL 、MyBatis -Plus 、Redis 、Spring Security 、JWT

项目 描述 ：

博睿 CRM 是为一家 私募投资公司 管理客户进行开发 ，平台管理投资者 信息 、基金产品、 交易记录和信息披露流程，

实现投资者从认购到资金分红的管理 。平台支持投资者偏好分析、收益模拟、风险评估及多级审批流程，提升基金管理

效率、合规水平和投资者满意度，为私募基金公司提供数据驱动的业务决策能力。

职责 描述 ：

• 集成 Spring Security+JWT ，分配角色和权限，确保用户有对应的权限；

• 投资者 管理模块 的开发 ，负责投资者的 投资偏好、历史认购记录 管理，并进行分类、标签化管理 ；

• 产品 管理模块 的开发，包括 产品信息、 份额记录 、收益统计等信息；

• 订单管理模块的开发，记录订单的生成，库存校验及状态跟踪，确保订单准确性 ；个人优势

• 拥有良好的团队协作与沟通能力，能够高效配合多部门完成项目交付

• 保持对新技术的持续学习和探索，紧跟行业发展，不断提升技术储备

• 注重代码质量与性能优化，致力于编写清晰、可维护且高效的系统代码
