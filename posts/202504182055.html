变量和运算符

北川2025-04-182025-04-18

1.八大基本类型及其字节数

  • byte（字节型）：占用 1 个字节（8 位），取值范围是 - 128 到 127。
  • short（短整型）：占用 2 个字节（16 位），取值范围是 - 32768 到 32767。
  • int（整型）：占用 4 个字节（32 位），取值范围是 - 2147483648 到 2147483647。
  • long（长整型）：占用 8 个字节（64 位），取值范围非常大，在定义 long 型变量时，通常在数字后面加 L 或 l，如 long num = 100L。
  • float（单精度浮点型）：占用 4 个字节（32 位），用于表示小数，在定义 float 型变量时，通常在数字后面加 F 或 f，如 float f = 3.14f。
  • double（双精度浮点型）：占用 8 个字节（64 位），也是用于表示小数，精度比 float 更高，定义时数字后面可不加 D 或 d，如 double d = 3.1415926。
  • char（字符型）：占用 2 个字节（16 位），用于存储单个字符，用单引号括起来，如 char c = 'A'。
  • boolean（布尔型）：在 Java 虚拟机规范中并没有明确规定其占用字节数，它只有两个字面值 true 和 false，通常用于逻辑判断。

2.基本类型间的大小排序

byte < short < int < long < float < double，char 类型在参与运算时会根据具体情况转换为 int 类型，boolean 类型不参与这种大小比较，因为它表示逻辑值，并非数值

3.自动类型转换和自动类型提升的触发时机

  • 自动类型转换：当把一个取值范围小的类型赋值给取值范围大的类型时，会发生自动类型转换。
  • 自动类型提升：在表达式中，如果操作数的类型不同，会发生自动类型提升。提升规则是将所有操作数的类型提升到与表达式中最高级别的操作数相同的类型

4.强转的语法

强制类型转换是将数据范围大的类型转换为数据类型范围小的类型

语法为：（目标类型）变量或表达式

5.什么是自动类型提升及提升范围

自动类型提升是指在表达式运算时，java会自动将参与运算的操作数转换为同一种范围更大的类型，以避免数据丢失或者精度损失。

  • 如果表达式中有 double 类型的操作数，所有操作数都会被提升为 double 类型。
  • 如果没有 double 类型，但有 float 类型，所有操作数会被提升为 float 类型。
  • 如果没有 float 和 double 类型，但有 long 类型，所有操作数会被提升为 long 类型。
  • 如果没有 long、float 和 double 类型，且操作数中有 byte、short 或 char 类型，这些类型会先被提升为 int 类型，然后整个表达式以 int 类型进行运算。

6.短路运算符和非短路运算符的区别

运算符 作用和含义
&&(逻辑与，表示并且) 全为真，才为真
||（逻辑或，表示或者） 一个为真，则为真
!（逻辑非，表示取反） 取相反值

1.短路运算符: && 和 ||

2.非短路运算符: & 和 | 特点为无论如何所有表达式都会执行

7.什么是短路机制及短路时机

短路机制: 当执行到可以奠定最终结果的表达式时, 后续表达式将不再执行

短路时机:

  • && : 执行到false时
  • || : 执行到true时

运算符 作用和含义
++ 数值+1
– 数值-1
  • 变量的值无论如何都会改变
  • 一元表达式的值: 符号在前, 结果为运算之后. 符号在后, 结果为运算之前
